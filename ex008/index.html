Listas

Listas ordenadas

Usa a tag ol (container) e os itens da lista com a tag li, sendo o seu fechamento opcional.

'type' = 1/A ou a/I ou i; usados para definir o que irá ordernar as listas(números, letras maiusculas ou minusculas, algarismos romanos maiusculos ou minusculos)

E o 'start' usado para indicar a partir de onde se iniciara a lista

Exemplo:

  1. Arroz
  2. Feijão
  3. Macarrão
  4. Farinha

Listas não ordenadas

Usa a tag ul e seus itens li, sendo essa não obrigatória o fechamento.

Os types disc, circle e square alteram as formas que acompanham os itens dessas listas

Exemplo:

  • Peixe
  • Carne moída
  • Batata frita
